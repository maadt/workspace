<ディレクトリの移動>
cd...change directory

cd .. ：一つ前のディレクトリに戻る
cd ディレクトリ名 ：現在のディレクトリの、子ディレクトリのみ指定できる
cd .. ：親ディレクトリへ移動
cd ~ ：ホームディレクトリへ移動
cd - ：直前のディレクトリに移動
cd ~/ディレクトリ名1/ディレクトリ名2/ディレクトリ名3 ：絶対パスを指定できる



<ファイルとディレクトリの表示>
ls...list

ls ： 現在のディレクトリ内のファイルとサブディレクトリの名前が表示されます。
ls -a ：現在のディレクトリ内のすべてのファイルとディレクトリ（隠しファイルを含む）を表示します。
ls -l ：ファイルやディレクトリの詳細情報（パーミッション、所有者、サイズなど）をリスト形式で表示します。
ls -la ：詳細情報と隠しファイルを一緒に表示します。
ls -lh ：ファイルサイズを読みやすい形式（KB、MBなど）で表示します。
ls -lt ：最終変更日時順にファイルを表示します。
